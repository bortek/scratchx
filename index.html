Walk Challenge

We challenge you to make Marty walk, but you're not allowed to just use the walk command!
You'll need to think about balance and how Marty moves, and you'll actually teach a robot to walk!

Intro with instructions

Once you're through the basics, try the following!:
  • Try using the "Move joint..." block to make your own movement, and define it in a function
  • Try making a step function that makes marty turn. See if you can get it to take an argument - in this case a number input - to vary the amount it turns
  • If the turn amount is too large, Marty might get into trouble. How can you make sure that doesn't happen?
